# ADR 0001: Stack tecnica

## Status

Aceita.

## Contexto

O trabalho exige uma aplicacao simples, funcional, publicavel e defendivel em apresentacao academica.

## Decisao

Usar:

- TypeScript
- Node.js
- Express.js
- EJS
- PostgreSQL
- Prisma
- Zod
- Vitest
- ESLint
- Prettier
- Render Web Service
- Cloudflare DNS e WAF quando houver dominio proprio

## Consequencias

A stack e suficiente para demonstrar aplicacao web, banco real, deploy em nuvem, DNS, firewall planejado e boas praticas de versionamento.
